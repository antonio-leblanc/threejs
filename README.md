## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

### Install dependencies (only the first time)
```
npm install
```

### Run the local server at localhost:8085

```
npm run dev
```

# Build and publish
```
npm run build

git subtree push --prefix dist origin gh-pages
```
